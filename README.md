# WinHide - 窗口管理器

一个功能强大的Windows窗口管理工具，支持窗口隐藏、显示、时间同步等功能。

## 功能特性

- **窗口管理**：支持窗口的隐藏、显示和切换
- **全局热键**：通过自定义热键快速隐藏/显示窗口
- **关键字自动选择**：添加关键字后，含关键字的窗口会自动选中
- **时间校准**：自动与NTP服务器同步系统时间，解决域服务器时间不准确问题
- **热键录制**：支持录制自定义热键（隐藏热键和显示热键分开录制）
- **进程白名单**：可设置进程白名单，过滤不需要管理的窗口
- **自动切换界面**：时间校准选项卡作为启动界面，30秒后自动切换回时间校准
- **开机自启动**：可选开机自启动功能
- **日志管理**：支持窗口操作日志和校时日志的开关控制

## 安装

### 依赖项

- Python 3.12+
- 必要的Python包：
  - PySide6
  - Pillow
  - pywin32
  - psutil
  - pynput

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/skybinbin3011365/WindowManager.git
cd WindowManager

# 安装依赖
pip install -r requirements.txt

# 运行主程序
python run_spec.py
```

### 打包发布

使用PyInstaller打包成可执行文件：

```bash
# 打包（使用统一的spec文件）
pyinstaller WinHide.spec

# 带控制台调试版本
pyinstaller WinHide.spec --console
```

打包后的文件位于 `dist/WinHide/` 目录。

## 使用指南

### 基本操作

1. **刷新窗口列表**：点击"刷新窗口"按钮更新窗口列表
2. **隐藏窗口**：选择窗口后点击"隐藏窗口"按钮
3. **显示窗口**：选择窗口后点击"显示窗口"按钮

### 热键操作

- **隐藏窗口热键**：默认 `鼠标中键 + 鼠标右键`
- **显示窗口热键**：默认 `Shift + 鼠标右键`（显示并立即最小化）

### 自定义热键

1. 切换到"设置"选项卡
2. 点击"隐藏热键录制"或"显示热键录制"按钮
3. 按下您想要设置的热键组合（2秒后自动完成录制）
4. 热键会自动保存

### 关键字管理

1. 在左侧"关键字管理"区域输入关键字
2. 点击"添加"按钮添加关键字
3. 窗口标题包含关键字的窗口会自动选中
4. 可删除选中的关键字或全部删除

### 时间校准

1. 程序启动后默认显示"时间校准"选项卡
2. 时间校准选项卡会立即初始化
3. NTP时间每分钟从网络更新一次
4. 本地时间显示每秒自动+1（不请求网络）
5. 时间差超出阈值时，自动校准系统时间
6. 可点击"立即更新NTP"手动更新NTP时间
7. **自动切换**：无论切换到哪个选项卡，30秒后会自动切换回时间校准选项卡

### 进程白名单

1. 在"设置"选项卡的"进程白名单"区域
2. 输入进程名（可带或不带.exe后缀）
3. 点击"添加"按钮添加到白名单
4. 白名单中的进程窗口不会显示在窗口列表中

### 日志控制

1. 在"设置"选项卡的"日志设置"区域
2. **窗口操作日志**：记录窗口隐藏/显示操作
3. **校时日志**：记录时间校准信息
   - 开启：显示所有时间调整记录
   - 关闭：仅显示大于1000ms的时间调整

### 时间校准设计原理

针对域服务器时间不准确的问题，本软件采用以下设计：

1. **软件内部时间管理**：
   - NTP时间：每分钟从网络更新一次
   - 本地时间：软件内部模拟，每秒自动+1，不请求网络

2. **时间差计算**：基于软件内部的两个时间计算和判断

3. **自动校准**：时间差超出阈值时，直接使用软件内部的NTP时间校准系统时间

这样既避免了频繁的网络请求，又确保了时间校准的准确性。

## 配置文件

程序使用`config/config.json`文件存储配置信息，包括：

- 热键设置（隐藏热键、显示热键）
- NTP服务器列表
- 进程白名单
- 开机自启动设置
- 日志显示选项
- 时间校准设置（自动校准、检查间隔、误差阈值）
- 关键字列表
- 选中的窗口

## 注意事项

- 时间校准功能需要管理员权限
- 热键可能与其他程序冲突，请选择合适的热键组合
- 在Windows 10/11系统上测试通过
- 建议使用管理员权限运行以获得完整功能
- 程序最小化到任务栏，不再使用系统托盘
