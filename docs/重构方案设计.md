# 窗口管理器重构方案设计

## 一、当前代码与用户文档的关键差异分析

### 1.1 窗口分类逻辑差异
**用户文档要求**：
- 严格按照Windows任务管理器标准区分"应用"和"后台进程"
- 应用 = 有可见窗口、可切换、非子窗口的顶层窗口
- 后台进程 = 无可见窗口的进程

**当前代码实现**：
- 使用`is_taskbar_window()`判断，不够精确
- 没有严格遵循Windows官方标准

### 1.2 设定窗口列表管理差异
**用户文档要求**：
- 设定窗口列表包含两类：关键字匹配的窗口 + 手动选中的窗口
- 状态管理：可见、隐藏、无效
- 进程监听：WMI实时监听新进程启动

**当前代码实现**：
- 简单的选中窗口集合，缺乏状态管理
- 没有进程监听机制

### 1.3 数据持久化差异
**用户文档要求**：
- 保存进程名、标题、句柄的完整字典
- 支持意外重启后的状态恢复

**当前代码实现**：
- 只保存进程名和标题，不保存句柄
- 状态恢复逻辑不完善

#### 2.1.2 窗口管理器核心类重构

```python
class WindowManager:
    """重构后的窗口管理器"""
    
    def __init__(self):
        self._target_windows: Dict[str, WindowEntry] = {}  # 进程名 -> 窗口条目
        self._process_monitor = ProcessMonitor()  # WMI进程监听器
        self._window_cache: Dict[int, WindowInfo] = {}     # 窗口缓存
        
    def refresh_windows(self):
        """按照Windows任务管理器标准刷新窗口"""
        # 1. 获取所有应用窗口（符合任务管理器标准）
        apps = self._get_taskmanager_apps()
        
        # 2. 获取后台进程
        background = self._get_background_processes()
        
        # 3. 更新设定窗口状态
        self._update_target_windows_status(apps + background)
        
    def _get_taskmanager_apps(self) -> List[WindowInfo]:
        """严格按照任务管理器标准获取应用窗口"""
        apps = []
        
        def enum_callback(hwnd, _):
            if self._is_taskmanager_app_window(hwnd):
                window_info = self._create_window_info(hwnd)
                apps.append(window_info)
            return True
            
        win32gui.EnumWindows(enum_callback, None)
        return apps
        
    def _is_taskmanager_app_window(self, hwnd: int) -> bool:
        """判断是否为任务管理器认定的应用窗口"""
        # 1. 必须可见
        if not win32gui.IsWindowVisible(hwnd):
            return False
            
        # 2. 必须是顶层窗口
        if win32gui.GetParent(hwnd) != 0:
            return False
            
        # 3. 必须有Alt+Tab可切换风格
        exstyle = win32gui.GetWindowLong(hwnd, win32con.GWL_EXSTYLE)
        if not (exstyle & win32con.WS_EX_APPWINDOW):
            return False
            
        # 4. 窗口标题不能为空
        title = win32gui.GetWindowText(hwnd).strip()
        if not title:
            return False
            
        return True
```

            self.target_table.insertRow(i)

### 2.3 进程监听机制

#### 2.3.1 WMI进程监听器

```python
class ProcessMonitor:
    """WMI进程监听器"""
    
    def __init__(self, target_processes: Set[str]):
        self.target_processes = target_processes
        self._running = False
        
    def start_monitoring(self):
        """开始监听进程创建事件"""
        self._running = True
        
        def monitor_thread():
            c = wmi.WMI()
            watcher = c.Win32_Process.watch_for("creation")
            
            while self._running:
                try:
                    new_process = watcher()
                    pid = new_process.ProcessId
                    name = new_process.Name.lower()
                    
                    if name in self.target_processes:
                        # 通知主线程处理新进程
                        self._on_new_process(pid, name)
                        
                except Exception as e:
                    logger.error("进程监听错误: %s", e)
                    break
                    
        threading.Thread(target=monitor_thread, daemon=True).start()
        
    def _on_new_process(self, pid: int, name: str):
        """处理新进程启动"""
        # 在主线程中更新UI和状态
        QMetaObject.invokeMethod(self, "update_target_window_status", 
                                Qt.QueuedConnection,
                                Q_ARG(int, pid), 
                                Q_ARG(str, name))
```

### 2.4 状态恢复机制

#### 2.4.1 意外重启后的状态恢复

```python
def restore_target_windows(self):
    """从配置恢复设定窗口状态"""
    if not self.config.target_windows:
        return
        
    for window_dict in self.config.target_windows:
        entry = WindowEntry(**window_dict)
        
        # 尝试查找对应的窗口
        hwnd = self._find_window_by_process(entry.process_name, entry.title)
        if hwnd:
            entry.hwnd = hwnd
            entry.state = WindowState.VISIBLE
            
            # 如果之前是隐藏状态，恢复隐藏
            if entry.state == WindowState.HIDDEN:
                self.hide_window(hwnd)
                
        else:
            entry.state = WindowState.INVALID
            
        self._target_windows[entry.process_name] = entry
```
