# WinHide - 窗口管理器

一个功能强大的 Windows 窗口管理工具，支持窗口隐藏、显示、时间校准等功能。

## 功能特性

- **窗口管理**：支持窗口的隐藏、显示和切换
- **全局热键**：通过自定义热键快速隐藏/显示窗口
- **关键字自动选择**：添加关键字后，含关键字的窗口会自动选中
- **热键录制**：支持录制自定义热键（隐藏热键和显示热键分开录制）
- **进程白名单**：可设置进程白名单，过滤不需要管理的窗口
- **开机自启动**：可选开机自启动功能
- **时间校准**：支持 NTP 时间同步，保持系统时间准确

## 安装

### 依赖项

- Python 3.12+
- 必要的Python包：
  - Pillow
  - pywin32
  - psutil
  - pynput

### 安装步骤

1. 克隆或下载项目到本地
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行主程序：
   ```bash
   python run.py
   ```

### 打包发布

使用PyInstaller打包成可执行文件：

```bash
# 无控制台窗口版本
pyinstaller windowmanager_noconsole.spec

# 带控制台窗口版本
pyinstaller windowmanager_console.spec
```

打包后的文件位于 `dist` 目录。

## 使用指南

### 基本操作

1. **刷新窗口列表**：点击"刷新窗口"按钮更新窗口列表
2. **隐藏窗口**：选择窗口后点击"隐藏窗口"按钮
3. **显示窗口**：选择窗口后点击"显示窗口"按钮

### 时间校准

1. 切换到"时间校准"选项卡
2. 点击"开始同步"按钮进行 NTP 时间同步
3. 同步完成后会显示时间偏移量
4. 点击"应用时间偏移"按钮可调整系统时间（需要管理员权限）

### 热键操作

- **隐藏窗口热键**：默认 `鼠标中键 + 鼠标右键`
- **显示窗口热键**：默认 `Shift + 鼠标右键`（显示并立即最小化）

### 自定义热键

1. 切换到"设置"选项卡
2. 点击"隐藏热键录制"或"显示热键录制"按钮
3. 按下您想要设置的热键组合（2秒后自动完成录制）
4. 热键会自动保存

### 关键字管理

1. 在左侧"关键字管理"区域输入关键字
2. 点击"添加"按钮添加关键字
3. 窗口标题包含关键字的窗口会自动选中
4. 可删除选中的关键字或全部删除

### 进程白名单

1. 在"设置"选项卡的"进程白名单"区域
2. 输入进程名（可带或不带.exe后缀）
3. 点击"添加"按钮添加到白名单
4. 白名单中的进程窗口不会显示在窗口列表中

## 配置文件

程序使用`config.json`文件存储配置信息，包括：

- 热键设置（隐藏热键、显示热键）
- 进程白名单
- 开机自启动设置
- 日志显示选项
- 关键字列表
- 选中的窗口

## 注意事项

- 热键可能与其他程序冲突，请选择合适的热键组合
- 在Windows 10/11系统上测试通过
- 建议使用管理员权限运行以获得完整功能
- 程序最小化到任务栏，不再使用系统托盘
